# Aerials Downloader for tvOS's Wallpapers

This Python script downloads Apple's aerial videos from tvOS and macOS, consolidates metadata from multiple sources, and provides localized asset information.

## What it does

1. Downloads tvOS 16, tvOS 13, and macOS 26 resource bundles from Apple
2. Parses `entries.json` from each source
3. Consolidates entries by asset ID, merging metadata from all sources
4. Loads localized asset names, category names, and points of interest from bundled strings files
5. Downloads the actual video files in your preferred quality

A Docker container can serve the files with CORS on `localhost:5050` — see the docker-compose setup below.

## Running this script

Expected environment variables:

| ENV VAR | Description | Default |
|---------|-------------|---------|
| `AERIAL_LANGUAGE` | ISO language code for localized asset names (e.g. `en`, `fr`, `de`, `ja`) | `en` |
| `VIDEO_QUALITY` | Video quality to download | `url-4K-SDR-240FPS` |
| `BW_LIMIT` | Limit video downloads to ~1 MB/s | not set (unlimited) |
| `SKIP_DOWNLOADS` | Skip downloading video files | not set |

**VIDEO_QUALITY options:** `url-1080-H264`, `url-1080-HDR`, `url-1080-SDR`, `url-4K-HDR`, `url-4K-SDR`

**Language codes:** The script supports any language code corresponding to a `.lproj` folder in the resources' `TVIdleScreenStrings.bundle` (e.g. `en`, `en_AU`, `fr`, `de`, `ja`, `zh_CN`, etc.). Falls back to English if unavailable.

Run:
```shell
AERIAL_LANGUAGE=fr VIDEO_QUALITY=url-4K-SDR python downloadAerials.py
```


## Running Docker

To serve the consolidated assets with CORS:

1. Run the downloader script above to generate the consolidated data and download videos
2. Start the container:

```shell
docker-compose up
```

The container uses a reverse proxy (see [nginx-static Docker Hub](https://hub.docker.com/r/flashspys/nginx-static/)) to serve files on `localhost:5050` with CORS headers enabled.
